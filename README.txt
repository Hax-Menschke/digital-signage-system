ShowTimerX – Lokale Installation mit XAMPP
1. Entpacke diesen Ordner nach /xampp/htdocs/
2. Öffne phpMyAdmin und erstelle eine Datenbank 'showtimerx'
3. Importiere die mitgelieferte SQL-Datei (in späterer Version enthalten)
4. Gehe im Browser zu http://localhost/ShowTimerX_Full/
5. Login mit Benutzer: admin / Passwort: admin
